#### How to use      
1. `git clone`
2. `npm i`
3. Make tea or coffee while downloading
4. `bower i`
5. Move the necessary JS modules to ./src/js/
6. `gulp build`
7. `gulp sprite`  
8. `npm install -g browser-sync`
9. `cd dist/`
10. `browser-sync start --server --files "**/*.*"`
11. `gulp`
