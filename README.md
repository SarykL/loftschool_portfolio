#### How to use      
1. `git clone`
2. `npm i`
3. Make tea or coffee while downloading
4. `bower i`
5. Move the necessary JS modules to ./src/js/
6. `gulp sprite`  
7. `gulp`
8. `npm install -g browser-sync` (Use another terminal tab)
9. `cd dist/`
10. `browser-sync start --server --files "**/*.*"`
